# Shared GitHub Templates

This repository provides the default issue forms and pull request
template for repositories in the `plessme` organization.

## Included Templates

- Story issue form
- Task issue form
- Bug issue form
- Pull request template

## Template Distribution

GitHub uses a public `.github` repository as the source for default
community health files across an organization. Repositories that do
not define their own issue or pull request templates inherit the
defaults from this repository.

If a repository includes its own templates, those local templates
take precedence for that repository.

## Notes

- Issue forms are stored in `.github/ISSUE_TEMPLATE/`
- The shared pull request template is stored in
  `.github/PULL_REQUEST_TEMPLATE.md`
- These templates are intentionally generic and contain no
  repository-specific workflow rules
